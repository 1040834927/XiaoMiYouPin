## 使用教程  

#### 1. 填写config.ini配置信息 
(1)`username`和`password`必填，小米账号和密码。

(2)`buy_time`,样例`2023-10-19 09:59:59.800`

(3)`act_id`可不填，程序会获取当天最新的商品id

(4)`DEFAULT_USER_AGENT`默认即可，也可去`https://www.bejson.com/network/useragent_rnd/ `选择一个喜欢的，注意选择安卓或者ios，不能选择电脑端
